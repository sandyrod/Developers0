HTML

Es un lenguaje de marcado de hipertexto, es decir un lenguaje basado en etiquetas

Cuando hablamos de hipertexto nos referimos a texto enriquecido con caracteristicas multimedia (imagenes, sonidos, videos) y de apariencia modificable (Color de fuente, tamaño entre otros).

Un archivo HTML es simplemente un archivo de texto que tiene las indicaciones de como debe mostrarse el contenido en un navegador, por lo cual no requerimos aplicaciones especializadas para escribirlo, basta con un editor de texto

Las Etiquetas

Son los componentes fundamentales del html, tambien se conocen como tag que es su nombre en ingles, son marcas que delimitan una region dentro del html por ejemplo: <p> es una etiqueta que indica el inicio de un parrafo </p> es otra etiqueta y esta indica el final del parrafo

Elemento

Un elemento consiste por lo general en una etiqueta de inicio y una etiqueta de cierre, por ejemplo este parrafo, o cada encabezado, son elementos formados por las etiquetas p y h2 respectivamente.

Atributo

Los atributos, son caracteristas adicionales de un elemento, cada elemento puede tener un grupo de atributos propios, como nombre, fuente, color entre otros. Por ejemplo, el elemento img que permite mostrar una imagen tiene un atributo llamado src a traves del cual podemos indicar la ubicacion de la imagen a mostrar

Estandares

Cuando construimos un sitio web y queremos que el mismo sea un producto de calidad, es necesario que cumplamos con estadares, en el caso del HTML estos estadares los dicta las W3C

Estructura de un HTML

Un archivo HTML bien formado contiene dos partes, un encabezado <head> y un cuerpo <body>
el head es lo primero que lee e interpreta el navegador, por lo tando alli suelen declararse algunas variables o importarse librerias, entre otras cosas, que veremos mas adelante
El elemento <body> o cuerpo del html es donde vamos a crear y dar forma a todo lo que queramos hacer visible en nuestra pagina web.
