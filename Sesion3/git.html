Comandos a utilizar en esta clase


Estaremos utilizando un repositorio creado en github

  1. clone: Permite clonar (copiar) un repositorio existente
    Ej:
    git clone https://github.com/sandyrod/Developers0.git
  2. add: Permite agregar archivos a un commit
    Ej:
    git add .
    git add NombreDelArchivo
    git add NombreDelDirectorio
  3. commit: Conjunto de cambios a enviar al repositorio
    Ej:
    git commit -m "descripcion de los cambios a realizar" Ver más
  4. push: Envia el commit al repositorio
    Ej:
    git push origin master
  5. pull: descarga los cambios del repositorio
    Ej:
    git pull origin master
  6. status: permite ver los cambios que hemos hecho en el repositorio local y la rama en la cual estamos
    Ej:
    git status
  7. branch: permite ver las ramas de nuestro repositorio
    Ej:
    git branch
  8. diff: permite ver las diferencias entre la version anterior y los cambios realizados
    Ej:
    git diff
    git diff NombreDelArchivo
